Worked Examples

Study step-by-step hand calculations for two numerical datasets to build solid mathematical intuition.

Example 1: Quadratic Interpolation (n = 3)

Construct a quadratic interpolating polynomial passing through the points: \((1, 3), (2, 11),\) and \((4, 47)\). Find the value of the function at \(x = 3\).

1
Identify Given Data Points
We have three coordinates: \[x_0 = 1, \; y_0 = 3\] \[x_1 = 2, \; y_1 = 11\] \[x_2 = 4, \; y_2 = 47\]
2
Compute Divided Differences

Compute the first-order divided differences:

\[f[x_0, x_1] = \frac{f(x_1) - f(x_0)}{x_1 - x_0} = \frac{11 - 3}{2 - 1} = 8\] \[f[x_1, x_2] = \frac{f(x_2) - f(x_1)}{x_2 - x_1} = \frac{47 - 11}{4 - 2} = \frac{36}{2} = 18\]

Compute the second-order divided difference:

\[f[x_0, x_1, x_2] = \frac{f[x_1, x_2] - f[x_0, x_1]}{x_2 - x_0} = \frac{18 - 8}{4 - 1} = \frac{10}{3} \approx 3.3333\]
3
Construct Divided Difference Table
\(x_i\) \(y_i\) \(1\text{st}\) \(2\text{nd}\)
1.0 3.0000 8.0000 3.3333
2.0 11.0000 18.0000 —
4.0 47.0000 — —
4
Build Interpolating Polynomial Formula
Using the first row values of the table as coefficients: \[P_2(x) = f[x_0] + f[x_0, x_1](x - x_0) + f[x_0, x_1, x_2](x - x_0)(x - x_1)\] \[P_2(x) = 3 + 8(x - 1) + 3.3333(x - 1)(x - 2)\]
5
Evaluate at Target Point (x = 3)
Substitute \(x = 3\) into the polynomial: \[P_2(3) = 3 + 8(3 - 1) + 3.3333(3 - 1)(3 - 2)\] \[P_2(3) = 3 + 8(2) + 3.3333(2)(1)\] \[P_2(3) = 3 + 16 + 6.6667 = 25.6667\]

Thus, the interpolated value at \(x = 3\) is approximately 25.6667.

Example 2: Cubic Interpolation (n = 4)

Determine the interpolating polynomial of degree 3 for the points: \((0, 2), (1, 3), (2, 12),\) and \((5, 147)\). Calculate \(P_3(3)\).

1
Identify Given Data Points
\[x_0 = 0, \; y_0 = 2\] \[x_1 = 1, \; y_1 = 3\] \[x_2 = 2, \; y_2 = 12\] \[x_3 = 5, \; y_3 = 147\]
2
Compute Divided Differences

1st order differences:

\[f[x_0, x_1] = \frac{3 - 2}{1 - 0} = 1\] \[f[x_1, x_2] = \frac{12 - 3}{2 - 1} = 9\] \[f[x_2, x_3] = \frac{147 - 12}{5 - 2} = \frac{135}{3} = 45\]

2nd order differences:

\[f[x_0, x_1, x_2] = \frac{9 - 1}{2 - 0} = \frac{8}{2} = 4\] \[f[x_1, x_2, x_3] = \frac{45 - 9}{5 - 1} = \frac{36}{4} = 9\]

3rd order difference:

\[f[x_0, x_1, x_2, x_3] = \frac{9 - 4}{5 - 0} = \frac{5}{5} = 1\]
3
Construct Divided Difference Table
\(x_i\) \(y_i\) \(1\text{st}\) \(2\text{nd}\) \(3\text{rd}\)
0.0 2.0000 1.0000 4.0000 1.0000
1.0 3.0000 9.0000 9.0000 —
2.0 12.0000 45.0000 — —
5.0 147.0000 — — —
4
Build Interpolating Polynomial Formula
Using the highlighted coefficients at the top of each column: \[P_3(x) = f[x_0] + f[x_0, x_1](x - x_0) + f[x_0, x_1, x_2](x - x_0)(x - x_1) + f[x_0, x_1, x_2, x_3](x - x_0)(x - x_1)(x - x_2)\] \[P_3(x) = 2 + 1(x - 0) + 4(x - 0)(x - 1) + 1(x - 0)(x - 1)(x - 2)\] \[P_3(x) = 2 + x + 4x(x - 1) + x(x - 1)(x - 2)\]
5
Evaluate at Target Point (x = 3)
Substitute \(x = 3\) into the simplified equation: \[P_3(3) = 2 + (3) + 4(3)(3 - 1) + 3(3 - 1)(3 - 2)\] \[P_3(3) = 2 + 3 + 4(3)(2) + 3(2)(1)\] \[P_3(3) = 5 + 24 + 6 = 35\]

Thus, the interpolated value at \(x = 3\) is exactly 35.
